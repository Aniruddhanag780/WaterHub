
# HydroTrack - Smart Hydration Tracker

This is a premium, intelligent water tracking application built with Next.js 15, Firebase, and Genkit.

## reCAPTCHA Setup
To ensure the authentication flow works correctly, please add the following domains to your reCAPTCHA v3 admin console:

- `localhost`
- `studio-1198494154-b6e32.firebaseapp.com`
- `studio-1198494154-b6e32.web.app`

## Getting Started
To run the development server:
```bash
npm run dev
```

The app will be available on port 9002.
